數據報表

全站遊戲數據頁

  1. 在遊戲列表頁點擊全站數據或是點擊左側選單加值行銷>樂遊戲>數據報表，進入全站遊戲數據頁
  2. 標題：全站遊戲數據
  3. 店家名稱：顯示店家名稱
  4. 次標題：遊戲與獎品
  5. 遊戲流量：上線中遊戲數、已下線遊戲數、總造訪次數、總造訪人數等資訊
  6. 遊戲列表 : 顯示上線中、已下線的遊戲資料，顯示的資料欄位為遊戲狀態、遊戲名稱、活動時間、造訪次數、造訪人數、獎品總數、未領取獎品數、已領取獎品數、獎品流向統計-自己領取、獎品流向統計-贈送他人
    1. 遊戲狀態、遊戲名稱是固定顯示，其他欄位可以左右滑動來檢視
    2. 欲察看個別遊戲的數據統計，可點擊欲察看的遊戲列任一位置，將另開新分頁並導到該遊戲的個別遊戲數據頁
    3. 遊戲數據列表中一次會顯示10筆資料，超過10筆資料可點擊上一頁、下一頁切換顯示
    4. 遊戲數據列表的排序規則是依照活動開始時間降冪排序

個別遊戲數據頁

  1. 在遊戲列表頁中點擊功能欄位中的 按鈕-遊戲數據統計 或是在全站遊戲數據頁中點擊欲察看的遊戲列進入個別遊戲數據頁
  2. 標題 : 個別遊戲數據
  3. 遊戲名稱 : 顯示目前察看的遊戲名稱
  4. 遊戲活動檔期 : 顯示遊戲的開始日期與結束日期
  5. 區塊切換 : 依遊戲類型不同會顯示不一樣的切換按鈕。轉盤遊戲僅顯示 按鈕-遊戲表現分析；而問答遊戲則會顯示 按鈕-遊戲表現分析、按鈕-作答結果分析，點擊可切換顯示不同的網頁
    6. 問答遊戲預設顯示遊戲表現分析頁
  6. 遊戲表現分析頁
    1. 標題1 : 遊戲造訪
    2. 遊戲造訪統計 : 提供總造訪次數、總造訪人數、折線圖 等資訊
      1. 折線圖有提供下拉選單-數據顯示區間，可選擇近7天、近30天、近90天、總活動期間 來顯示區間統計資料
      2. 折線圖預設數據顯示區間為近7天
      3. 滑鼠游標停在折線圖上會顯示當天的統計資料
    3. 標題2 : 獎品兌換
    4. 獎品兌換列表 : 顯示所有獎品的詳細資料，顯示的資料欄位為獎品名稱、獎品類型、獎品總數、未領取獎品數、已領取獎品數、獎品流向統計-自己領取、獎品流向統計-贈送他人
      5. 資料排序是依照建立遊戲時新增獎品項目的順序顯示
