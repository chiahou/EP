數據報表

全站遊戲數據頁

  1. 在 網頁-遊戲列表頁 點擊 Link-全站數據 或是點擊 左側Sidebar-加值行銷>遊戲平台 中的 Link-數據報表 進入 網頁-全站遊戲數據頁
  2. 標題：Label-全站遊戲數據
  3. 店家名稱：顯示店家名稱
  4. 次標題：Label-遊戲與獎品
  5. 遊戲流量：上線中遊戲數、已下線遊戲數、總造訪次數、總造訪人數等資訊
  6. 遊戲列表 : 顯示上線中、已下線的遊戲資料，顯示的資料欄位為【遊戲狀態】欄位、【遊戲名稱】欄位、【活動時間】欄位、【造訪次數】欄位、【造訪人數】欄位、【獎品總數】欄位、【未領取獎品數】欄位、【已領取獎品數】欄位、【獎品流向統計-自己領取】欄位、【獎品流向統計-贈送他人】欄位
    1. 【遊戲狀態】欄位、【遊戲名稱】欄位 是固定顯示，其他欄位可以左右滑動來檢視
    2. 欲察看個別遊戲的數據統計，可點擊欲察看的遊戲列任一位置，將另開新分頁並導到該遊戲的 網頁-個別遊戲數據頁
    3. Table-遊戲列表 一次顯示10筆資料，超過10筆資料可點擊 Link-上一頁、Link-下一頁 切換顯示
    4. Table-遊戲列表 的排序規則是依照活動開始時間降冪排序

個別遊戲數據頁

  1. 在 網頁-遊戲列表頁 中點擊功能欄位中的 按鈕-遊戲數據統計 或是在 網頁-全站遊戲數據頁 中點擊欲察看的遊戲列進入 網頁-個別遊戲數據頁
  2. 標題 : Label-個別遊戲數據
  3. 遊戲名稱 : 顯示目前察看的遊戲名稱
  4. 遊戲活動檔期 : 顯示遊戲的開始日期與結束日期
  5. 區塊切換 : 依遊戲類型不同會顯示不一樣的切換按鈕。轉盤遊戲僅顯示 按鈕-遊戲表現分析；而問答遊戲則會顯示 按鈕-遊戲表現分析、按鈕-作答結果分析，點擊可切換顯示不同的網頁
    6. 問答遊戲預設顯示 網頁-遊戲表現分析頁
  6. 網頁-遊戲表現分析頁
    1. 標題1 : 「遊戲造訪」
    2. 遊戲造訪統計 : 提供【總造訪次數】欄位、【總造訪人數】欄位、曲線圖 等資訊
      1. 曲線圖 有提供 下拉選單-數據顯示區間，可選擇【近7天】選項、【近30天】選項、【近90天】選項、【總活動期間】選項 來顯示區間統計資料
      2. 曲線圖 預設數據顯示區間為【近7天】選項
      3. 滑鼠指標停在曲線上會顯示當天的統計資料
    3. 標題2 : Label-獎品兌換
    4. 獎品兌換列表 : 顯示所有獎品的詳細資料，顯示的資料欄位為 【獎品名稱】欄位、【獎品類型】欄位、【獎品總數】欄位、【未領取獎品數】欄位、【已領取獎品數】欄位、【獎品流向統計-自己領取】欄位、【獎品流向統計-贈送他人】欄位
      5. 資料排序是依照建立遊戲時新增獎品項目的順序顯示
