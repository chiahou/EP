遊戲狀態

  1. 草稿新增一個遊戲，但尚未完成，狀態為「草稿」。遊戲列表顯示此遊戲，可以編輯、刪除。
  2. 完成編輯：編輯完成一個遊戲，狀態為「完成編輯」。遊戲列表顯示此遊戲，可以編輯、刪除，或付費開啟遊戲，付費流程請參考付費 。
  3. 付費完成為遊戲付費成功後，狀態為「付費完成」。 遊戲列表顯示此遊戲，可以檢視遊戲設定（但不能編輯），或取消退費，退款流程請參考退費。
  4. 待上線為遊戲付費成功後，稍等幾小時，狀態為「待上線」。 遊戲列表顯示此遊戲，可以分享遊戲連結開始宣傳遊戲、檢視遊戲設定（但不能編輯），或點擊取消退費，開始退款流程。退款流程請參考退費。
  5. 已退費將遊戲取消退費成功後，狀態為「已退費」
  6. 上線中活動開始時間到，遊戲狀態變為「上線中」，就能在樂遊戲前台正式遊玩。 遊戲列表顯示此遊戲，可以查看遊戲設定（但不能編輯），或查看遊戲數據。
  7. 已下線活動結束時間到，遊戲狀態變為「已下線」，就不能在樂遊戲前台遊玩。 遊戲列表顯示此遊戲，可以查看遊戲數據。
  8. 遊戲不存在若樂天市場變更或刪除你選擇的遊戲方案，且當下你的遊戲狀態為「草稿」或「完成編輯」時，遊戲狀態變為「遊戲不存在」（若變更或刪除遊戲方案當下你已為遊戲付費成功，則遊戲狀態不變）。 此遊戲狀態下，要麻煩你先刪除該遊戲，再另外新增一款新的遊戲。

付費

  1. 網頁入口
    1. 路徑1
      1. 在新增遊戲頁面完成第五步後
      2. 點擊開啟遊戲，顯示付費提醒彈窗
      3. 點擊確定付費
        4. 點擊確定付費後會做以下判斷 :
        • 判斷活動開始日是否晚於目前時間+24小時
        • 獎品生效時間是否晚於或等於活動開始日
        • 獎品失效時間是否晚於活動結束日+24小時
        • 商店是否已啟用。 如果上述情況皆有符合，會將網頁導至 網頁-付款頁；若有不符合，會顯示 彈窗 並告知如何做後續處理。
    2. 路徑2
      1. 點擊遊戲列表頁中的付費開啟 ，網頁導到新增遊戲第五步：完成頁
        2. 當遊戲狀態為「完成編輯」時才會顯示付費開啟
      2. 點擊開啟遊戲，顯示付費提醒彈窗
      3. 點擊確定付費
  2. 介面說明
    1. 標題：付款
    2. 標題：信用卡資訊
    3. 標題：卡片詳細資料
    4. 信用卡卡號：在信用卡卡號輸入信用卡卡號
      • 信用卡卡號 預設顯示 範例「1234-5678-9012-1234」
      • 輸入錯誤的信用卡卡號時會在信用卡卡號下方顯示 錯誤訊息「請輸入有效的 信用卡卡號」
    5. 持卡人姓名 : 在持卡人姓名 輸入持卡人姓名
      • 持卡人姓名下方有 提醒文字「如果使用台灣發行的信用卡付款，請用中文輸入持卡人姓名
      • 輸入不符合格式的持卡人姓名時會在持卡人姓名下方顯示 錯誤訊息「請僅使用字母 (半形或全形)、數字 (0-9)、空格和下列特殊字元: ' . -」
    6. 信用卡有效期限 : 在 下拉選單-月、下拉選單-年 選擇信用卡的有效期限
      1. 選擇錯誤的信用卡有效期限時會在下拉選單-月、下拉選單-年 下方顯示 錯誤訊息「請輸入有效的 信用卡有效期限。」
    7. 安全碼 : 在安全碼 輸入信用卡安全碼
      • 安全碼右側有一圖示，滑鼠指標移動到這會跳出安全碼說明
      • 輸入不符合格式的安全碼時會在安全碼下方顯示錯誤訊息「請輸入有效的 安全碼。」
    8. 標題 : 發票資訊(資料同合約內容，不得自行變更)
    9. 發票類別 : 選擇發票類別
      10. 目前僅有三聯可選擇，日後會提供其他選項
    10. 統一編號 :統一編號會自動帶入店家登記的統一編號，請注意此處不可編輯
    11. 發票抬頭 :發票抬頭會自動帶入店家登記的發票抬頭，請注意此處不可編輯
    12. 按鈕-取消 : 點擊後顯示取消付款確認彈窗
      • 內文 : 「你是否要取消付款？」
      • 按鈕-否 : 點擊後關閉 彈窗-取消付款確認
      • 按鈕-是，我要取消 : 點擊後關閉 彈窗-取消付款確認 並將網頁導回遊戲列表
    13. 按鈕-付款 : 當所有欄位皆有資料後，點擊後開始執行付款動作
      1. 若付款失敗，將網頁導至付款失敗頁
      2. 若付款成功，將網頁導至付款成功頁

退費

  1. 從左側選單進入遊戲列表頁
  2. 前置條件：
    1. 目前時間必須在遊戲開始時間前10分鐘之前
    2. 遊戲狀態為付費完成/待上線
  3. 操作步驟
    1. 點擊遊戲列表中欲退費遊戲的付費操作按鈕取消退費，顯示取消活動彈窗並選擇確認退款
    2. 在取消活動原因輸入取消活動的原因
      3. 取消活動原因為必填欄位，最多可輸入100個字
    3. 點擊進行退款，顯示退款成功彈窗
      4. 取消活動原因尚未輸入任何字時，無法點擊進行退款
    4. 點擊知道了，關閉所有彈窗
    5. 遊戲狀態 變更為「已退費」
